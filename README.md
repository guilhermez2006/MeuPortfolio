# 📱 Meu Portfólio

Bem-vindo ao meu portfólio pessoal! Este é um site responsivo desenvolvido com **HTML5**, **CSS3** e **JavaScript**, apresentando meus projetos e habilidades.

## 🚀 Características

- **Design Moderno**: Interface limpa com animações suaves
- **100% Responsivo**: Adapta-se a todos os tamanhos de tela (mobile, tablet, desktop)
- **Animações Interativas**: Efeitos parallax, scroll suave e transições elegantes
- **Projetos em Destaque**: 6 projetos principais com modais detalhados
- **Partículas Animadas**: Fundo dinâmico com particles.js
- **Links para Redes Sociais**: GitHub, Instagram e LinkedIn

## 📂 Estrutura do Projeto

```
MeuPortfolio/
├── assets/
│   ├── css/
│   │   ├── style.css              # Estilos principais
│   │   └── responsiveness.css     # Media queries
│   ├── js/
│   │   └── script.js              # Lógica interativa
│   ├── img/                       # Imagens do portfólio
│   └── fonts/                     # Fontes customizadas
├── index.html                     # Arquivo principal
├── README.md                      # Este arquivo
└── LICENSE                        # Licença MIT
```
